# Smart Temperature and Gas Detection System Using Arduino UNO

## Overview

This project monitors temperature and gas leakage using an Arduino UNO, DHT11 sensor, and MQ Gas Sensor. The system displays real-time readings on a 16×2 I2C LCD and activates an LED and buzzer whenever hazardous conditions are detected.

## Features

- Real-time temperature monitoring
- Gas leakage detection
- 16×2 I2C LCD display
- LED and buzzer alert system
- Four operating modes:
  - Safe Zone
  - Temperature Warning
  - Gas Warning
  - Danger

## Hardware Components

- Arduino UNO
- DHT11 Temperature and Humidity Sensor
- MQ Gas Sensor
- 16×2 I2C LCD
- LED
- Active Buzzer
- Breadboard
- Jumper Wires
